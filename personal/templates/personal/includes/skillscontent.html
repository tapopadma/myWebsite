Fingerprint Recognition System Development:

This was my final year project. It's objective is to design a fingerprint recognition based biometric system development. For this MATLAB tool boxes played a vital role. Both for the GUI design and the image processing algorithm implementation MATLAB is good enough. The GUI's purpose is to let the user load image from local disk onto it and find the matching score for each of the existing image from the local database with respect to the loaded image and finally give a verdict on SUCCESSFUL or UNSUCCESSFUL MATCH.
